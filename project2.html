To-Do CRUD Application.

A simple To-Do List with modifiable functionalities like create, read, update and delete to keep user updated on the work he/she has to do.

The project was developed using Bootstrap, PHP and MySQL.

  • Utilized CRUD principles to optimize the efficiency and reliability of to-do list applications
  • Used SQL database for storing items.
  • If a user enters wrong task then the task can be modified later using the CRUD feature. Also tasks can be deleted after completing their deadline.
