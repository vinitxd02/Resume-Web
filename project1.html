Reminder through WhatsApp notification.

  • Demo:
  • Source code

The system helps you with reminding the work you have set at the appropriate time by sending a message in your WhatsApp Number.

It is a full stack project developed with the help of MongoDB, ExpressJs, ReactJs, NodeJS along with the help of Twilio API.

  • Created the UI using ReactJs and handles storing the data using MongoDB database.
  • Used various ReactJs features and NodeJs to establish proper connectivity.
  • Once the tasks are set the API, Twilio handles reminding the user their task on time.
