House Price Prediciton System.

  • Demo:
  • Source code

A system to predict the price for a flat in various localities so the user can plan out the budget while planning their hoem in the city of Mumbai.

Along with the locality, the system predicts the price with addon features like area, baths, car parking, etc.

  • The dataset was cleaned and pre-processed using various EDA methods which included outlier removal, removing noisy data,etc.
  • Various regression algorithms were used in training data. We used LASSO regression for this project as it gave the maximum accuracy.
  • The website was created using Django, a python framework.
